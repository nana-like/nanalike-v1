  • Branding

    브랜딩

    신생 스타트업과 신규 서비스의 브랜딩을 작업한 경험이 있습니다. 로고나 폰트 등의 디자인에 그치지 않고, 기업의 철학을 다듬고 내부 세미나를 여는 등 브랜드의 내실을 다지는 작업을 했습니다. 서비스의 경우, 사용자 시나리오와 전략을 함께 논의하고 그에 맞는 비전과 철학을 만들어 공유했습니다.

  • UI Design

    UI 디자인

    프로토타입 툴을 활용해 웹사이트의 UI를 디자인하는 것이 가능합니다. 기획 단계에서도 사용자의 동선이나 정보의 중요도 등을 고려한 디자인을 제시하여, 더 나은 사용자 경험을 만들기 위해 고민했습니다.

  • UX Writing

    UX 라이팅

    짧게는 버튼 문구부터 길게는 상세 페이지 소개글까지, 유저와 브랜드가 맞닿아 있는 모든 부분의 카피라이팅을 담당했습니다. 브랜드가 사용자와 만날 때 어떤 식으로 얘기하면 좋은지 구성원에게 공유하는 톤앤보이스 가이드를 작성하기도 했습니다.
