  • Branding

    브랜딩

    초기 스타트업과 서비스의 브랜딩을 작업한 경험이 있습니다. 로고나 폰트 같은 디자인뿐만 아니라, 브랜드의 기반을 다지는 작업도 함께 진행했습니다. 사용자 시나리오와 전략을 바탕으로 서비스의 비전과 철학을 다듬고, 내부 세미나를 열어 공유하는 등 브랜드의 내실을 다졌습니다.

  • UI Design

    UI 디자인

    프로토타입 툴을 활용해 웹 서비스의 UI를 디자인할 수 있습니다. 단순히 기획안 구현에서 끝나지 않고, 사용자의 동선이나 정보의 중요도 등을 고려한 디자인을 제시해 더 나은 서비스를 만들기 위해 고민합니다.

  • UX Writing

    UX 라이팅

    짧게는 버튼 문구부터 길게는 상세 페이지 소개글까지, 유저와 브랜드가 맞닿아 있는 모든 부분의 카피라이팅을 담당했습니다. 안정된 UX를 위해 브랜드가 사용자에게 어떻게 얘기하면 좋은지 가이드를 만들기도 합니다.
